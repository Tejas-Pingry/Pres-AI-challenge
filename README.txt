# VETERAN BOT (OLLAMA)

## SETUP

	1. Install Ollama (https://ollama.com/download)
	2. Clone this repository:
```bash
git clone https://github.com/<your-username>/veteran-bot.git
cd veteran-bot
	3. Create the model from Modelfile (ollama create veteran-bot -f Modelfile)
	4. Run the model (ollama run veteran-bot)
